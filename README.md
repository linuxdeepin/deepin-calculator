# Deepin Calculator

Deepin calculator is an easy to use calculator for ordinary users.

## Dependencies

* sudo apt install libdtkwidget-dev libqt5svg5-dev

## Installation

* mkdir build
* cd build
* qmake ..
* make

## Usage

* ./deepin-calculator
